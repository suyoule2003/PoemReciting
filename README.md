PoemReciting
===================================
简介
------------------------------------------------------------
一个简单的Python小程序，用于输出随机的挖空诗句。

用法
------------------------------------------------------------
数据库格式： <br>
    [<br> 
    诗的题目 <br>
    诗的年代 <br>
    诗的作者 <br>
    一小句，一小句，一小句。 <br>
    一小句，一小句。 <br>
    …… <br>
    一小句，一小句。 <br>
    ] <br>
运行[poem.py](https://github.com/suyoule2003/PoemReciting/blob/master/poem.py)即可。
